HTML

HTML, acrónimo en inglés de HyperText Markup Language ('lenguaje de marcado de hipertexto'), hace referencia al lenguaje de marcado utilizado en la creación de páginas web.

HTML es un lenguaje de marcado que posibilita definir la estructura de nuestro documento mediante etiquetas.

Historia de HTML

A principios de 1990, Tim Berners-Lee define por fin el HTML como un subconjunto del conocido SGML y crea algo más valioso, incluso, el World Wide Web.

Versiones de HTML

Tim Berbers-Lee definió la primera versión de HTML en 1991.

En la actualidad la última versión es HTML 5.
